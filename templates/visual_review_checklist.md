# 视觉审查清单

## 封面

- [ ] 标题是否一眼可读。
- [ ] 标题是否占据足够视觉权重。
- [ ] 封面是否只有一个核心钩子。
- [ ] 主视觉是否服务标题。
- [ ] 背景是否干扰文字。
- [ ] 是否避免廉价蓝紫科技风。

## 内页

- [ ] 每页是否只承担一个主要任务。
- [ ] 是否有清晰标题、副标题、正文和注释层级。
- [ ] 是否存在大段文字堆积。
- [ ] 是否用结构图、卡片、对比、标注提升理解效率。
- [ ] 页面之间是否有递进关系。
- [ ] 页面之间是否有节奏变化。

## 风格一致性

- [ ] 主色、辅助色、强调色是否统一。
- [ ] 字体是否统一。
- [ ] 角标、页码、标签样式是否统一。
- [ ] 主风格和辅助风格是否互相支持。
- [ ] 是否有某一页突然像另一个模板。

## 小红书适配

- [ ] 手机端是否能读清。
- [ ] 是否适合滑动阅读。
- [ ] 是否有收藏价值。
- [ ] 是否有评论话题。
- [ ] 是否有关注理由。

## 反模式检查

- [ ] 不像普通 PPT。
- [ ] 不像廉价 AI 模板。
- [ ] 不信息过载。
- [ ] 不伪高级。
- [ ] 不文字不可读。
