# 09. 高级商业提案风

## 风格定义

- 适合内容：企业服务、义乌商户、AI+产业、文旅方案、招商路演。
- 不适合内容：纯个人情绪、轻松娱乐内容。
- 视觉气质：稳重、专业、有信任感、有商业审美。
- 配色：深蓝黑 / 暖白 / 金色 / 银灰 / 少量亮色。
- 字体：稳重无衬线，数据数字可加粗。
- 构图：大标题 + 数据卡片 + 行业场景图 + 流程图。
- 常用元素：贸易网络、地图、数据卡片、商业关键词。
- 图像提示词模板：3:4 竖版商业提案页，深蓝黑背景，左侧大标题和商业关键词，右侧数据卡片或行业网络图，金色细线强调信任感。
- 负面提示词：土味招商海报、金色过度、股票软件感。
- 示例标题类型：“AI 如何帮义乌商户多卖一亿单”。

## 图像提示词模板

- 适合：企业服务、产业方案、AI+商业。
- 不适合：个人情绪、轻娱乐。
- 视觉关键词：商务、信任、数据卡片、行业场景。
- 版式结构：左标题，右数据 / 场景 / 流程。

```text
生成一张 3:4 竖版小红书商业提案页，主题是「[主题]」。深蓝黑背景，左侧放大标题和 2 个商业关键词，右侧使用数据卡片、行业场景图或流程图构成主视觉。配色为深蓝黑、暖白、银灰，少量暖金细线增强信任感。字体稳重清晰，标题有商业发布会气质。页面要专业可信，不像土味招商。
负面：土味招商海报、金色过度、股票软件感、廉价商务模板。
```
