# 16. 高级极简黑金风

## 风格定义

- 适合内容：商业、品牌、成交、课程、咨询、企业方案。
- 不适合内容：轻松教程、年轻化工具清单、实验复盘。
- 视觉气质：高端、稳重、贵气、克制。
- 配色：黑 / 暖金 / 米白 / 深灰。
- 字体：稳重粗标题，金色细线做结构。
- 构图：大标题、金色细线、品牌图形、少量数据。
- 常用元素：金属细线、品牌符号、材质光影、数据点。
- 图像提示词模板：3:4 竖版黑金商业封面，黑色背景，暖金细线构成结构，标题大而克制，少量数据卡片增强价值感。
- 负面提示词：暴发户金色、复杂纹样、土味课程海报。
- 示例标题类型：“从内容效率，到商业增长”。

## 图像提示词模板

- 适合：商业、品牌、课程、咨询。
- 不适合：轻教程、工具清单。
- 视觉关键词：黑金、品牌感、细线、贵气克制。
- 版式结构：大标题 + 金色结构线 + 少量数据。

```text
生成一张 3:4 竖版小红书黑金商业封面，主题是「[主题]」。黑色背景，暖金细线构成结构框架，标题大而克制，副标题和数据卡片数量很少。可以加入高级金属光泽或品牌符号，但不能复杂。整体高端、稳重、像企业级方案封面。
负面：暴发户金色、复杂纹样、土味课程海报、金色铺满。
```
