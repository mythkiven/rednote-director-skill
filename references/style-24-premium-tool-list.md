# 24. 高级工具清单风

## 风格定义

- 适合内容：软件推荐、AI 工具合集、效率工具、工作流。
- 不适合内容：强情绪观点、单一产品发布会。
- 视觉气质：清单感、收藏感、秩序感、实用。
- 配色：深色或浅色均可，每个工具统一卡片承载。
- 字体：工具名清晰，使用场景一句话。
- 构图：工具卡片、分类标签、推荐指数、使用场景。
- 常用元素：图标、评分、分类、标签、列表编号。
- 图像提示词模板：3:4 竖版高级工具清单页，统一卡片网格展示 6-8 个工具，每张卡有图标、名称、使用场景和推荐标签，整体留白清晰。
- 负面提示词：图标模糊、卡片拥挤、像应用商店截图。
- 示例标题类型：“我的 Vibe Coding 工具箱”。

## 图像提示词模板

- 适合：软件推荐、AI 工具合集、效率工具。
- 不适合：强情绪观点、单一产品发布。
- 视觉关键词：工具卡片、分类标签、推荐指数、收藏感。
- 版式结构：统一卡片网格。

```text
生成一张 3:4 竖版小红书高级工具清单页，主题是「[主题]」。用统一卡片网格展示 6-8 个工具，每张卡有图标占位、工具名、使用场景一句话和推荐标签。顶部有大标题和分类标签，背景深色或浅灰都可，卡片间距一致，信息清晰，适合收藏。
负面：图标模糊、卡片拥挤、应用商店截图感、文字太小。
```
