# 20. 情绪共鸣 / 夜间独白风

## 风格定义

- 适合内容：个人思考、创业感悟、AI 时代焦虑、成长内容。
- 不适合内容：高密度教程、产品参数页。
- 视觉气质：安静、深夜、克制、真诚。
- 配色：深蓝黑 / 灰白 / 暖光点缀。
- 字体：标题不过度大，留白承载情绪。
- 构图：大面积暗色，一句话居中或偏左，少量环境光。
- 常用元素：窗边光、桌面、微弱光源、短句。
- 图像提示词模板：3:4 竖版夜间独白页，深蓝黑大面积留白，左下或中部一句短句，远处有克制暖光，整体安静。
- 负面提示词：伤感非主流、强滤镜、鸡汤字卡。
- 示例标题类型：“AI 时代，焦虑是正常的”。

## 图像提示词模板

- 适合：个人思考、创业感悟、AI 焦虑。
- 不适合：高密度教程、产品页。
- 视觉关键词：深夜、留白、暖光、短句。
- 版式结构：大面积暗色 + 一句话。

```text
生成一张 3:4 竖版小红书夜间独白页，主题是「[主题]」。深蓝黑背景，大面积留白，中部偏左放一句短句标题占位，远处有克制暖光或桌面微光。字体干净无衬线，正文很少，整体安静、真诚、有深夜思考感。
负面：伤感非主流、强滤镜、鸡汤字卡、复杂装饰。
```
