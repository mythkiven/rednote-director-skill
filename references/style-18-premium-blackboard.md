# 18. 课程讲义 / 高级黑板风

## 风格定义

- 适合内容：教程、知识科普、课程系列、Day 系列。
- 不适合内容：产品发布、商业品牌页。
- 视觉气质：系统、教学、清晰、专业。
- 配色：深色黑板 / 米白字 / 绿色或黄色重点。
- 字体：粗标题 + 清晰正文，少量手写标注但不幼稚。
- 构图：章节编号、知识点卡片、公式感结构。
- 常用元素：章节号、公式箭头、重点圈画、知识卡片。
- 图像提示词模板：3:4 竖版高级黑板讲义页，深色黑板质感，顶部章节标题，中部三块知识点卡片，绿色重点标注，字迹清晰。
- 负面提示词：儿童黑板报、粉笔灰过重、卡通插画。
- 示例标题类型：“Day 01｜为什么必须学 Vibe Coding”。

## 图像提示词模板

- 适合：教程、知识科普、Day 系列。
- 不适合：产品发布、商业封面。
- 视觉关键词：章节编号、知识卡片、公式结构、重点标注。
- 版式结构：顶部章节，中部知识点，底部总结。

```text
生成一张 3:4 竖版小红书高级黑板讲义页，主题是「[主题]」。深色黑板质感背景，顶部有章节编号和大标题，中部用 3 个知识点卡片或公式结构解释内容，少量绿色或黄色重点圈画。字体清晰，不要幼稚手写风，整体像高级课程讲义。
负面：儿童黑板报、粉笔灰过重、卡通插画、字太小。
```
