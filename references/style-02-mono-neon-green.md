# 02. 黑白灰 + 荧光绿冲击风

## 风格定义

- 适合内容：强观点、反常识、技术布道、个人观点、爆款封面。
- 不适合内容：温柔生活方式、治愈系、产品细节展示。
- 视觉气质：犀利、克制、有攻击性、有设计感。
- 配色：黑 / 白 / 灰为主，荧光绿只用于关键词、线条、按钮、编号。
- 字体：超大黑体标题，英文用紧凑无衬线。
- 构图：巨大标题、强留白、一个绿色关键词作为视觉焦点。
- 常用元素：警示线、编号、极简几何、短横线、标签。
- 图像提示词模板：3:4 竖版，黑白灰极简冲击封面，标题占画面近半，只有一个荧光绿关键词和细线强调，背景干净，有压迫感留白。
- 负面提示词：多色渐变、荧光色铺满、装饰过多、文字拥挤。
- 示例标题类型：“你不是不会用 AI，你是不会拆任务”“不要再让 AI 直接写页面了”。

## 图像提示词模板

- 适合：强观点、反常识、技术布道、爆款封面。
- 不适合：柔和产品页、长教程内页。
- 视觉关键词：巨大标题、强留白、黑白灰、荧光绿关键词。
- 版式结构：标题占 40%，一个绿色关键词做视觉锚点。

```text
生成一张 3:4 竖版小红书封面，主题是「[主题]」。黑白灰极简冲击风，背景为纯黑或近黑，标题区占画面 40%，使用超大粗黑体，标题中只有一个关键词用荧光绿突出。画面只保留少量细线、编号和几何切割，形成犀利、克制、有攻击性的视觉。正文区域很少，底部放一行小注释占位。整体要有强对比和高级留白，不要花哨。
负面：多色渐变、荧光绿铺满、装饰过多、表情包感、文字拥挤。
```
