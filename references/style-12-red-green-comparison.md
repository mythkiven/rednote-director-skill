# 12. 红绿对错对比风

## 风格定义

- 适合内容：错误示范 vs 正确示范、Prompt 优化、设计改版。
- 不适合内容：纯情绪、品牌宣言、产品美图。
- 视觉气质：清晰、有判断力、实用、收藏导向。
- 配色：错误区低饱和红，正确区低饱和绿，背景可黑可白。
- 字体：对比标签清楚，短句为主。
- 构图：左右对比、上下对比、Before / After。
- 常用元素：Bad / Good、Mistake / Better、对勾、叉号、标注线。
- 图像提示词模板：3:4 竖版对错对比页，左侧低饱和红错误示范，右侧低饱和绿正确示范，中间清晰分隔线，顶部一句结论。
- 负面提示词：红绿刺眼、像考试卷、文字密集。
- 示例标题类型：“错误的 Prompt 长这样”。

## 图像提示词模板

- 适合：Prompt 优化、设计改版、错误示范。
- 不适合：情绪文、品牌宣言。
- 视觉关键词：Bad / Good、Before / After、低饱和红绿。
- 版式结构：左右对比，顶部结论。

```text
生成一张 3:4 竖版小红书对错对比页，主题是「[主题]」。顶部放一句结论标题占位，下面左右两栏对比：左侧错误示范使用低饱和红和暗色卡片，右侧正确示范使用低饱和绿和清晰卡片。中间用细分隔线，卡片内保留短句占位。整体实用、清晰、有收藏价值。
负面：红绿刺眼、考试卷感、文字密集、卡片拥挤。
```
