# 13. 赛博档案 / 黑客文件风

## 风格定义

- 适合内容：Agent、自动化、代码工具、实验复盘、技术探索。
- 不适合内容：商业招商、温柔生活方式、白底杂志文。
- 视觉气质：神秘、实验、硬核、酷。
- 配色：黑色 / 暗绿 / 灰白，少量终端绿。
- 字体：无衬线 + 等宽字体点缀。
- 构图：档案袋、文件编号、终端窗口、系统日志。
- 常用元素：扫描线、编号、代码片段、状态条。
- 图像提示词模板：3:4 竖版黑客档案封面，黑色背景，终端窗口和文件编号，暗绿状态灯，标题像机密实验记录但文字清晰。
- 负面提示词：低质黑客电影海报、乱码文字、过度故障效果。
- 示例标题类型：“我让 Codex 生了一个小 Codex”。

## 图像提示词模板

- 适合：Agent、自动化、技术实验复盘。
- 不适合：商业招商、温柔教程。
- 视觉关键词：终端、文件编号、系统日志、暗绿。
- 版式结构：档案标题 + 终端窗口 + 状态条。

```text
生成一张 3:4 竖版小红书赛博档案封面，主题是「[主题]」。黑色背景，画面像一份技术实验档案：文件编号、终端窗口、系统日志、扫描线、暗绿色状态灯。标题区放在左上，使用粗体无衬线，局部使用等宽字体作为技术注释。整体神秘、硬核、实验感强，但文字区域必须清晰。
负面：低质黑客电影海报、乱码文字、过度故障效果、绿色铺满。
```
