# 21. 数据报告 / 趋势洞察风

## 风格定义

- 适合内容：行业趋势、研究结论、市场洞察、案例分析。
- 不适合内容：纯情绪、封面爆梗、生活种草。
- 视觉气质：权威、理性、可信、有信息密度。
- 配色：白底或深色底，蓝灰 / 绿色 / 黑色。
- 字体：大数字突出，结论句清晰。
- 构图：大数字、图表、趋势线、结论卡片、来源区。
- 常用元素：折线图、柱状图、结论标签、引用来源。
- 图像提示词模板：3:4 竖版数据趋势报告页，顶部结论，大数字作为视觉中心，中部简洁趋势图，底部来源注释，配色克制。
- 负面提示词：图表太密、假数据堆叠、财经软件感。
- 示例标题类型：“2026 年，普通人的代码杠杆来了”。

## 图像提示词模板

- 适合：行业趋势、市场洞察、研究结论。
- 不适合：纯情绪、生活种草。
- 视觉关键词：大数字、趋势线、图表、来源区。
- 版式结构：顶部结论，中部图表，底部来源。

```text
生成一张 3:4 竖版小红书数据趋势报告页，主题是「[主题]」。顶部是一句结论标题，中间用一个大数字和一张简洁趋势图作为视觉中心，底部有来源注释区。配色为白底或深色底，蓝灰、绿色和黑色为主。图表简洁可信，信息密度高但不拥挤。
负面：图表太密、假数据堆叠、财经软件感、字号过小。
```
