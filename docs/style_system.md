# 小红书高级图文视觉风格系统

## 默认审美基线

- 默认画幅：3:4 竖版，小红书手机端优先。
- 页面气质：高级、设计师审美、科技感、杂志感、极简但有冲击力。
- 信息结构：每页只承担一个主要任务，标题先抓住人，内页再给结构和证据。
- 字体：中文用思源黑体、阿里巴巴普惠体、HarmonyOS Sans、苹方风格；英文用 Inter、SF Pro、Helvetica Neue 风格。
- 色彩：黑、深灰、白、银灰为基础，可少量使用荧光绿、薄荷绿、电光蓝、亮橙、低饱和红。
- 禁止：廉价蓝紫渐变、随机霓虹线条、文字过小、PPT bullet 堆砌、塑料质感、儿童卡通感、巨大页码抢戏。

## 风格组合机制

- 封面可以更冲击，内页可以更理性：封面用反差冲击封面风，内页用架构图 / 系统拆解风，结尾用个人品牌宣言风。
- 教程类内容要增强收藏感：主风格 Notion 高级卡片风，辅助风格课程讲义 / 高级黑板风。
- AI 技术观点要避免太冷：主风格深色科技杂志风，辅助风格情绪共鸣 / 夜间独白风。
- 商业提案要避免太网感：主风格高级商业提案风，辅助风格全球贸易网络风。
- 产品展示要避免太 PPT：主风格高级电商详情页风，辅助风格极简产品发布会风。

## 24 种内置风格库

### 1. 深色科技杂志风

- 适合内容：AI、Vibe Coding、Agent、工具、认知升级、方法论、技术趋势。
- 不适合内容：强生活化、亲子、甜美种草、低信息密度情绪文。
- 视觉气质：高级、冷静、理性、科技、专业。
- 配色：黑色、深灰、银白，少量荧光绿 / 电光蓝。
- 字体：粗黑体大标题 + Inter 小号英文注释。
- 构图：大标题占 30%-45%，背景细网格、结构线、玻璃面板。
- 常用元素：代码窗口、Agent 网络、抽象芯片、数据流、状态标签。
- 图像提示词模板：3:4 竖版深色科技杂志封面，左上大标题区，右下抽象 Agent 网络或代码窗口，细网格背景，银白文字，少量荧光绿强调，清晰留白。
- 负面提示词：廉价蓝紫渐变、随机霓虹、复杂无意义背景、文字变形、过小文字。
- 示例标题类型：“AI 写代码，总翻车？”“普通人必须学会 Vibe Coding”“别再把 AI 当搜索框用了”。

### 2. 黑白灰 + 荧光绿冲击风

- 适合内容：强观点、反常识、技术布道、个人观点、爆款封面。
- 不适合内容：温柔生活方式、治愈系、产品细节展示。
- 视觉气质：犀利、克制、有攻击性、有设计感。
- 配色：黑 / 白 / 灰为主，荧光绿只用于关键词、线条、按钮、编号。
- 字体：超大黑体标题，英文用紧凑无衬线。
- 构图：巨大标题、强留白、一个绿色关键词作为视觉焦点。
- 常用元素：警示线、编号、极简几何、短横线、标签。
- 图像提示词模板：3:4 竖版，黑白灰极简冲击封面，标题占画面近半，只有一个荧光绿关键词和细线强调，背景干净，有压迫感留白。
- 负面提示词：多色渐变、荧光色铺满、装饰过多、文字拥挤。
- 示例标题类型：“你不是不会用 AI，你是不会拆任务”“不要再让 AI 直接写页面了”。

### 3. Notion 高级卡片风

- 适合内容：工具推荐、方法清单、工作流、资料整理、知识管理。
- 不适合内容：强冲突观点、宏大商业叙事、情绪独白封面。
- 视觉气质：干净、理性、可收藏、温和专业。
- 配色：米白 / 浅灰 / 黑字 / 少量蓝灰或绿色。
- 字体：中等字重无衬线，标题清楚但不夸张。
- 构图：多张卡片组合，类似知识库页面。
- 常用元素：标签、折叠块、引用块、目录、checkbox、分割线。
- 图像提示词模板：3:4 竖版知识库卡片页面，浅灰背景，多张圆角很小的白色卡片，清晰标题区、标签区、步骤区，像高级 Notion 模板但更有杂志感。
- 负面提示词：卡片过多、字号过小、花哨贴纸、过度阴影。
- 示例标题类型：“我把 AI 工作流整理成了这 6 个模块”。

### 4. 液态玻璃 / 弥散极光风

- 适合内容：产品发布、个人品牌、AI 应用、未来感选题、封面。
- 不适合内容：高密度教程、严肃商业报告、需要强可读表格的内容。
- 视觉气质：未来、柔和、高级、梦幻但不俗。
- 配色：深色底 + 低饱和蓝紫 / 绿色 / 粉橙弥散光。
- 字体：细腻无衬线，标题需放在清晰暗区。
- 构图：柔和光晕背景，前景玻璃卡片，40% 以上留白。
- 常用元素：液态玻璃、半透明面板、光晕、折射边缘。
- 图像提示词模板：3:4 竖版，深色背景上的低饱和弥散光，前景半透明玻璃面板，标题区域干净高对比，光效只服务主视觉。
- 负面提示词：彩虹渐变铺满、强紫蓝俗气光效、文字被光效干扰。
- 示例标题类型：“我做了一个自己的 AI 工作台”。

### 5. 极简产品发布会风

- 适合内容：产品展示、工具介绍、个人项目发布、MVP 展示。
- 不适合内容：长教程、复杂观点辩论、情绪共鸣文。
- 视觉气质：Apple-like、克制、高端、发布会感。
- 配色：黑白灰，单一高亮色。
- 字体：大标题简洁，卖点小标签。
- 构图：中心产品主视觉，大标题极简，少量卖点标签。
- 常用元素：设备 mockup、产品截图、光影台面、hero section。
- 图像提示词模板：3:4 竖版产品发布会海报，中心展示软件界面或产品，黑白灰背景，单一强调色，标题短而大，卖点用小标签排列。
- 负面提示词：PPT 列表、过度装饰、廉价样机、反光过强。
- 示例标题类型：“这个小工具，解决了我 80% 的重复工作”。

### 6. 反差冲击封面风

- 适合内容：爆款封面、反常识观点、争议选题、强情绪开头。
- 不适合内容：温和工具清单、品牌长线建设首图、严肃白皮书。
- 视觉气质：直接、尖锐、强对比、一眼停住。
- 配色：黑白强对比，红 / 绿 / 黄作为警示色。
- 字体：粗体大字，短句优先。
- 构图：左右对比、上下对比、错误 vs 正确、旧世界 vs 新世界。
- 常用元素：叉号、对勾、撕裂分割、警示框、对比箭头。
- 图像提示词模板：3:4 竖版强对比封面，左右两栏形成冲突，左侧错误状态暗红，右侧正确状态低饱和绿，中央用大标题压住视觉。
- 负面提示词：表情包感、低俗营销、颜色过饱和、巨大感叹号。
- 示例标题类型：“99% 的人用错了 AI”。

### 7. 架构图 / 系统拆解风

- 适合内容：Agent 架构、产品系统、API Contract、工程化、复杂方法论。
- 不适合内容：低信息密度情绪文、纯审美分享。
- 视觉气质：专业、清晰、系统性强。
- 配色：深色底 / 白色卡片 / 绿色或蓝色连线。
- 字体：标题粗，节点标签中等，注释小而清晰。
- 构图：模块化节点、箭头、层级结构、输入 / 处理 / 输出。
- 常用元素：节点、连接线、流程箭头、状态灯、分层面板。
- 图像提示词模板：3:4 竖版系统架构图，顶部一句结论，中部模块节点和箭头，底部输出结果区，深色背景，白色卡片，绿色连线。
- 负面提示词：箭头混乱、节点过多、文字过小、伪 3D 图标。
- 示例标题类型：“一个 Agent 应用到底由什么组成？”。

### 8. 手机截图改造风

- 适合内容：App 工作流、桌面整理、工具教程、真实案例展示。
- 不适合内容：无真实截图的抽象观点、复杂商业方案。
- 视觉气质：真实、可模仿、生活化但高级。
- 配色：背景克制，截图为主视觉。
- 字体：标注文字清晰，标签短。
- 构图：截图保持真实比例，周围加标签、箭头、编号。
- 常用元素：手机框、标注线、编号、功能标签。
- 图像提示词模板：3:4 竖版手机截图讲解页，中间是真实比例手机截图，四周留出标注区，用细线和编号解释功能区，背景低调高级。
- 负面提示词：拉伸截图、夸张手机边框、密集小字、贴纸感。
- 示例标题类型：“我是这样整理手机桌面的”。

### 9. 高级商业提案风

- 适合内容：企业服务、义乌商户、AI+产业、文旅方案、招商路演。
- 不适合内容：纯个人情绪、轻松娱乐内容。
- 视觉气质：稳重、专业、有信任感、有商业审美。
- 配色：深蓝黑 / 暖白 / 金色 / 银灰 / 少量亮色。
- 字体：稳重无衬线，数据数字可加粗。
- 构图：大标题 + 数据卡片 + 行业场景图 + 流程图。
- 常用元素：贸易网络、地图、数据卡片、商业关键词。
- 图像提示词模板：3:4 竖版商业提案页，深蓝黑背景，左侧大标题和商业关键词，右侧数据卡片或行业网络图，金色细线强调信任感。
- 负面提示词：土味招商海报、金色过度、股票软件感。
- 示例标题类型：“AI 如何帮义乌商户多卖一亿单”。

### 10. 全球贸易网络风

- 适合内容：义乌、外贸、跨境电商、全球化、供应链、物流。
- 不适合内容：个人成长、单一工具教程。
- 视觉气质：宏大、国际化、连接感、商业科技。
- 配色：深色底，低饱和金色 / 蓝色 / 白色。
- 字体：国际化商务无衬线。
- 构图：右侧地图或全球网络，左侧留标题区。
- 常用元素：点状地图、虚线连接、货船、飞机、集装箱图标。
- 图像提示词模板：3:4 竖版全球贸易网络海报，深色背景，点状世界地图，连接线从义乌发散到全球，左侧大标题区清晰。
- 负面提示词：地图太花、图标卡通、连接线过密、蓝紫科技模板。
- 示例标题类型：“义乌，正在进入 AI 外贸时代”。

### 11. 高级白底杂志风

- 适合内容：认知文章、审美分享、个人观点、理性内容。
- 不适合内容：强科技产品发布、暗黑实验感内容。
- 视觉气质：干净、编辑感、杂志专栏、克制。
- 配色：白 / 米白 / 黑 / 灰 / 少量红色或蓝色。
- 字体：杂志感无衬线，标题有字重对比。
- 构图：大量留白、大标题、小注释、细线分割。
- 常用元素：专栏编号、细线、引用、极简图标。
- 图像提示词模板：3:4 竖版白底杂志专栏页，大量留白，左侧大标题，右下小图形或注释，细线组织信息层级。
- 负面提示词：空得没有重点、字体太细、PPT 标题页感。
- 示例标题类型：“AI 时代，别再躲在岗位之后”。

### 12. 红绿对错对比风

- 适合内容：错误示范 vs 正确示范、Prompt 优化、设计改版。
- 不适合内容：纯情绪、品牌宣言、产品美图。
- 视觉气质：清晰、有判断力、实用、收藏导向。
- 配色：错误区低饱和红，正确区低饱和绿，背景可黑可白。
- 字体：对比标签清楚，短句为主。
- 构图：左右对比、上下对比、Before / After。
- 常用元素：Bad / Good、Mistake / Better、对勾、叉号、标注线。
- 图像提示词模板：3:4 竖版对错对比页，左侧低饱和红错误示范，右侧低饱和绿正确示范，中间清晰分隔线，顶部一句结论。
- 负面提示词：红绿刺眼、像考试卷、文字密集。
- 示例标题类型：“错误的 Prompt 长这样”。

### 13. 赛博档案 / 黑客文件风

- 适合内容：Agent、自动化、代码工具、实验复盘、技术探索。
- 不适合内容：商业招商、温柔生活方式、白底杂志文。
- 视觉气质：神秘、实验、硬核、酷。
- 配色：黑色 / 暗绿 / 灰白，少量终端绿。
- 字体：无衬线 + 等宽字体点缀。
- 构图：档案袋、文件编号、终端窗口、系统日志。
- 常用元素：扫描线、编号、代码片段、状态条。
- 图像提示词模板：3:4 竖版黑客档案封面，黑色背景，终端窗口和文件编号，暗绿状态灯，标题像机密实验记录但文字清晰。
- 负面提示词：低质黑客电影海报、乱码文字、过度故障效果。
- 示例标题类型：“我让 Codex 生了一个小 Codex”。

### 14. 未来实验室风

- 适合内容：AI 实验、产品原型、未来工具、概念验证。
- 不适合内容：轻教程、清单、传统商业介绍。
- 视觉气质：前沿、探索、实验、研究感。
- 配色：深灰 / 冷白 / 电光蓝 / 荧光绿。
- 字体：技术感无衬线，状态标签用等宽。
- 构图：实验台、模块面板、数据观察窗、原型图。
- 常用元素：观察窗、模块编号、实验状态、数据面板。
- 图像提示词模板：3:4 竖版未来实验室页面，深灰背景，多块实验观察面板，中心是 AI 原型模块，顶部实验标题，底部状态读数。
- 负面提示词：科幻电影海报感、元素太满、文字不可读。
- 示例标题类型：“我做了一个 AI 工作台实验”。

### 15. 设计师灵感板风

- 适合内容：审美教程、页面改版、封面风格选择、设计技巧。
- 不适合内容：严肃数据报告、强商业转化页。
- 视觉气质：灵感、审美、拼贴、设计过程感。
- 配色：黑白灰为基础，可加入低饱和色块。
- 字体：标题克制，标签精确。
- 构图：多图拼贴、参考图、色卡、字体样张、组件拆解。
- 常用元素：Moodboard、色卡、注释线、组件样张。
- 图像提示词模板：3:4 竖版设计灵感板，多张参考卡片有秩序地拼贴，包含色卡、字体样张、组件拆解，背景克制。
- 负面提示词：杂乱 scrapbook、贴纸感、颜色过多。
- 示例标题类型：“如何让页面看起来更高级？”。

### 16. 高级极简黑金风

- 适合内容：商业、品牌、成交、课程、咨询、企业方案。
- 不适合内容：轻松教程、年轻化工具清单、实验复盘。
- 视觉气质：高端、稳重、贵气、克制。
- 配色：黑 / 暖金 / 米白 / 深灰。
- 字体：稳重粗标题，金色细线做结构。
- 构图：大标题、金色细线、品牌图形、少量数据。
- 常用元素：金属细线、品牌符号、材质光影、数据点。
- 图像提示词模板：3:4 竖版黑金商业封面，黑色背景，暖金细线构成结构，标题大而克制，少量数据卡片增强价值感。
- 负面提示词：暴发户金色、复杂纹样、土味课程海报。
- 示例标题类型：“从内容效率，到商业增长”。

### 17. 软件界面 UI 风

- 适合内容：AI 产品、SaaS、工具、工作台、自动化流程。
- 不适合内容：没有产品界面支撑的情绪文。
- 视觉气质：产品感、真实、可落地、专业。
- 配色：深色 UI 或浅色 UI，强调层级真实。
- 字体：界面字体清晰，数字和状态标签精确。
- 构图：Dashboard、侧边栏、卡片、操作面板、数据面板。
- 常用元素：导航栏、状态栏、图表、任务卡片、Agent 卡片。
- 图像提示词模板：3:4 竖版软件界面展示页，真实 dashboard 布局，左侧侧边栏，中间任务卡片，右侧状态面板，标题区域与界面分层明确。
- 负面提示词：伪 UI、按钮乱码、过度玻璃、界面细节不可读。
- 示例标题类型：“我给自己搭了一个 AI 操作系统”。

### 18. 课程讲义 / 高级黑板风

- 适合内容：教程、知识科普、课程系列、Day 系列。
- 不适合内容：产品发布、商业品牌页。
- 视觉气质：系统、教学、清晰、专业。
- 配色：深色黑板 / 米白字 / 绿色或黄色重点。
- 字体：粗标题 + 清晰正文，少量手写标注但不幼稚。
- 构图：章节编号、知识点卡片、公式感结构。
- 常用元素：章节号、公式箭头、重点圈画、知识卡片。
- 图像提示词模板：3:4 竖版高级黑板讲义页，深色黑板质感，顶部章节标题，中部三块知识点卡片，绿色重点标注，字迹清晰。
- 负面提示词：儿童黑板报、粉笔灰过重、卡通插画。
- 示例标题类型：“Day 01｜为什么必须学 Vibe Coding”。

### 19. 个人品牌宣言风

- 适合内容：自我介绍、账号定位、观点输出、个人 IP。
- 不适合内容：工具清单、产品参数说明。
- 视觉气质：有态度、有识别度、克制、锋利。
- 配色：黑白灰 + 少量个人品牌色。
- 字体：大标题像宣言，副信息像名片。
- 构图：人物或抽象头像，一句宣言，个人标签，小号英文注释。
- 常用元素：签名、头像、域名、个人标签、细线框。
- 图像提示词模板：3:4 竖版个人品牌宣言海报，黑白灰背景，左侧一句大宣言，右侧抽象头像或个人符号，下方个人标签和签名。
- 负面提示词：鸡汤海报、明星写真感、过度自拍。
- 示例标题类型：“AI 时代，我选择公开构建”。

### 20. 情绪共鸣 / 夜间独白风

- 适合内容：个人思考、创业感悟、AI 时代焦虑、成长内容。
- 不适合内容：高密度教程、产品参数页。
- 视觉气质：安静、深夜、克制、真诚。
- 配色：深蓝黑 / 灰白 / 暖光点缀。
- 字体：标题不过度大，留白承载情绪。
- 构图：大面积暗色，一句话居中或偏左，少量环境光。
- 常用元素：窗边光、桌面、微弱光源、短句。
- 图像提示词模板：3:4 竖版夜间独白页，深蓝黑大面积留白，左下或中部一句短句，远处有克制暖光，整体安静。
- 负面提示词：伤感非主流、强滤镜、鸡汤字卡。
- 示例标题类型：“AI 时代，焦虑是正常的”。

### 21. 数据报告 / 趋势洞察风

- 适合内容：行业趋势、研究结论、市场洞察、案例分析。
- 不适合内容：纯情绪、封面爆梗、生活种草。
- 视觉气质：权威、理性、可信、有信息密度。
- 配色：白底或深色底，蓝灰 / 绿色 / 黑色。
- 字体：大数字突出，结论句清晰。
- 构图：大数字、图表、趋势线、结论卡片、来源区。
- 常用元素：折线图、柱状图、结论标签、引用来源。
- 图像提示词模板：3:4 竖版数据趋势报告页，顶部结论，大数字作为视觉中心，中部简洁趋势图，底部来源注释，配色克制。
- 负面提示词：图表太密、假数据堆叠、财经软件感。
- 示例标题类型：“2026 年，普通人的代码杠杆来了”。

### 22. 故事漫画分镜风

- 适合内容：用故事讲复杂概念、AI 与人类关系、讽刺漫画、轻量表达。
- 不适合内容：严肃商业提案、产品卖点页。

- 配色：黑白线稿，少量强调色。
- 字体：对白极少，标题清楚。
- 构图：3-4 格分镜，主角动作清晰，留白多。
- 常用元素：分镜框、简洁人物、隐喻道具、短对白。
- 图像提示词模板：3:4 竖版高级黑白分镜漫画，三格结构，用简洁线稿表达 AI 使用误区，只有一个强调色，文字极少。
- 负面提示词：儿童卡通、表情包、夸张漫画字、杂乱背景。
- 示例标题类型：“拿着芯片当石头用”。

### 23. 高级电商详情页风

- 构图：产品大图、卖点标签、使用场景、包装展示。

### 24. 高级工具清单风

- 适合内容：软件推荐、AI 工具合集、效率工具、工作流。

- 字体：工具名清晰，使用场景一句话。

- 示例标题类型：“我的 Vibe Coding 工具箱”。
