# 示例输出方案：义乌商户为什么必须看懂 AI

主题：义乌商户为什么必须看懂 AI？

# 选题判断

- 输入类型：商业趋势选题。
- 内容类型：商业趋势 + 产业方案 + AI 普及。
- 目标受众：义乌商户、跨境电商从业者、外贸服务商、本地产业服务机构。
- 内容机会：把“AI 很远”的抽象话题，转成“获客、选品、内容、客服、履约效率”的商业现实。

# 核心观点

- 核心观点一句话：义乌商户看懂 AI，不是为了追热点，而是为了把选品、内容、获客和外贸沟通变成新的效率系统。
- 传播钩子：下一轮小商品竞争，不只拼货源，也拼内容和响应速度。
- 收藏理由：读者可以收藏 AI 改造商户经营的 5 个环节。
- 评论区话题：你最想先用 AI 改造哪个环节？

# 风格判断报告

## 内容类型判断

- 当前内容属于：商业提案型 + 趋势洞察型 + 方法论型。
- 判断依据：
  - 面向商户和产业，必须建立信任感。
  - 内容需要解释趋势，也要落到业务环节。
  - 读者需要看到“这跟我的生意有关”，而不是泛泛讲 AI。

## 传播目标判断

- 主要目标：建立专业度 + 促进转化。
- 次要目标：促成收藏 + 引发咨询。

## 读者情绪判断

- 封面：感到相关，意识到 AI 已经进入生意链路。
- 第 2-3 页：产生紧迫感，但不是被吓到。
- 第 4-6 页：觉得实用，开始联想到自己的店铺。
- 结尾页：愿意留言自己的行业或商品类目。

## 信息密度判断

- 当前信息密度：高。
- 推荐版式：高级商业提案风 + 全球贸易网络风 + 数据报告 / 趋势洞察风 + 架构图 / 系统拆解风。
- 注意事项：避免互联网黑话，避免过度科技感；必须让商户读懂。

## 风格匹配结果

- 主风格：高级商业提案风。
- 辅助风格：全球贸易网络风；数据报告 / 趋势洞察风；架构图 / 系统拆解风。
- 不建议使用的风格：赛博档案 / 黑客文件风、液态玻璃 / 弥散极光风。
- 选择理由：商户场景需要可信、稳重、可转化；过度黑客或梦幻会显得不落地。
- 风格关键词：深蓝黑、贸易网络、数据卡片、流程图、暖金细线。
- 配色建议：深蓝黑、暖白、银灰、低饱和金色。
- 字体建议：稳重无衬线，标题大，数据数字加粗。
- 构图建议：封面用全球连接，内页用业务链路和数据卡片拆解。

# 推荐风格方案

## 方案 A：点击优先

- 风格组合：全球贸易网络风 + 反差冲击封面风。
- 适合原因：用“义乌正在进入 AI 外贸时代”制造趋势感。
- 风险：如果冲突太强，会像标题党。

## 方案 B：收藏优先

- 风格组合：架构图 / 系统拆解风 + 数据报告 / 趋势洞察风。
- 适合原因：能把经营链路拆清楚，便于收藏和转发。
- 风险：封面需要加强视觉钩子。

## 方案 C：高级感 / 商业转化优先

- 风格组合：高级商业提案风 + 全球贸易网络风。
- 适合原因：专业、可信、适合引导咨询或项目合作。
- 风险：文字需要更短，否则容易像路演 PPT。

我的推荐：
选择方案 C，并在第 4-6 页加入方案 B 的结构拆解。

# 逐页视觉规划摘要

## Page 01｜封面

- 页面目标：建立趋势感和商业相关性。
- 视觉构图：左侧大标题，右侧义乌连接全球的点状地图。
- 主视觉：从义乌发散到海外市场的贸易网络。
- 图像生成提示词：生成一张 3:4 竖版小红书商业趋势封面，主题是「义乌商户必须看懂 AI」。深蓝黑背景，右侧是点状世界地图和贸易连接线，连接线从义乌发散到全球；左侧保留大标题安全区，使用稳重无衬线字体，配色为暖白、银灰和少量低饱和金色。画面要像高级商业提案封面，不要土味招商。
- 负面提示词：廉价蓝紫科技风、金色铺满、地图过密、股票软件感。

## Page 04｜框架页

- 页面目标：把 AI 应用落到经营链路。
- 视觉构图：五个节点串联：选品、商品图、内容、询盘、客服。
- 图像生成提示词：生成一张 3:4 竖版小红书系统拆解页，主题是「AI 改造义乌商户的 5 个经营环节」。深色背景，中部是从选品到客服的五节点流程图，每个节点用商务卡片承载，节点之间用低饱和金色细线连接。顶部放结论标题区，底部放“先从内容素材开始”的注释区。整体专业、清晰、适合商户收藏。
- 负面提示词：箭头混乱、节点太多、PPT 流程图、文字太小。
